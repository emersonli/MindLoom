# MindLoom v1.0.0 - Phase 2 功能发布

**发布日期**: 2026-04-02  
**Git 提交**: f92e78d

---

## 🎉 新功能

### 🌓 暗色/明亮主题切换 (P2-05)
- 支持暗色/明亮/系统偏好三种模式
- localStorage 持久化
- 平滑过渡动画
- 全组件主题适配

### 📜 笔记版本历史 (P2-01)
- 自动版本保存
- 版本列表查看
- 版本预览功能
- 一键恢复历史版本
- 带确认对话框防止误操作

### 📅 每日笔记模板 (P2-03)
- "每日笔记"快捷按钮
- 自动日期标题 (YYYY-MM-DD 星期 X)
- 预设模板 (待办/学习/想法/事项)
- 重复笔记检查

### ⌨️ 全局快捷键系统 (P2-04)
- Ctrl+N: 新建笔记
- Ctrl+S: 保存笔记
- Ctrl+F: 搜索
- Ctrl+G: 知识图谱
- Ctrl+?: 快捷键帮助
- 编辑器内格式化快捷键 (Ctrl+B/Ctrl/I 等)

### 🕸️ 知识图谱可视化 (P2-02)
- D3.js 力导向图
- 拖拽节点交互
- 滚轮缩放 (0.5x - 4x)
- 点击节点跳转笔记
- 双向链接显示
