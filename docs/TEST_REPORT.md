## 🔴 已知问题

### 问题 1: 笔记删除后列表未刷新

**严重程度**: 中
**模块**: 笔记管理
**描述**: 删除笔记后，笔记列表未自动刷新，需要手动刷新页面
**状态**: 待修复
**修复计划**: 在删除后调用 setNotes 强制更新

### 问题 2: 搜索索引未实时更新

**严重程度**: 低
**模块**: 全文搜索
**描述**: 新建笔记后，搜索索引未立即更新，需要切换页面才能搜索到
**状态**: 待修复
**修复计划**: 在创建笔记后立即调用 searchService.addNote()

### 问题 3: 加密功能测试环境限制

**严重程度**: 低
**模块**: 安全加密
**描述**: Web Crypto API 在部分测试环境可能不可用
**状态**: 已知限制
**说明**: 生产环境需确保 HTTPS 访问
